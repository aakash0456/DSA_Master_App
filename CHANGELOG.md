# Changelog

## 1.2.0 — 2026-07-18
- New Pattern Playbook: 21 algorithmic patterns (Two Pointers, Sliding Window, Prefix Sum, Hashing, Stack, Monotonic Stack, Fast & Slow Pointers, In-place Reversal, Cyclic Sort, Merge Intervals, Binary Search, BFS, DFS/Recursion, Topological Sort, Union-Find, Trie, Heap/Top-K, Two Heaps, Greedy, DP, Bit Manipulation)
- Each pattern teaches recognition: statement signals, anti-signals ("probably NOT this pattern when..."), a Kotlin template, and linked practice problems
- "How to pick a pattern" decision checklist at the top of the playbook
- Entry banner on the Problems tab; database schema v3

## 1.1.0 — 2026-07-18
- Problems are now organized by algorithmic pattern with colored section headers
- New pattern-recognition training on every problem: "signals in the statement" clues and a numbered decision walkthrough from observation to chosen algorithm — shown before hints and solution
- More colorful UI: accent-colored topic cards, difficulty chips (green/amber/red), pattern tags, tinted progress stat pills
- Database schema v2 (adds pattern fields; dev builds recreate and reseed the database)

## 1.0.0
- Initial release: 18 topics with lessons, built-in flashcard deck, SM-2 review system,
  quizzes, coding problems, search, streaks + heatmap, settings, daily reminders.
